# SideNote AI Mobile Companion

Expo/React Native scaffold for iOS and Android companion apps.

## Scope

The first mobile app is a companion for meeting notes, search, export, and review. It should not claim hidden/background call recording. Mac local capture remains in `macos/SideNoteRecorder`.

## IDs

- iOS bundle ID: `ai.helix.sidenote`
- Android package: `ai.helix.sidenote`

## Commands

```bash
cd /Users/jaimehenriques/Helix-Labs/apps/sidenote-ai/mobile
npm install
npm run typecheck
npm run ios
npm run android
```
